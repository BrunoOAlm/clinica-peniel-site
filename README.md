# 🏥 Clínica Peniel - Site Institucional

Projeto de site institucional desenvolvido para a Clínica Peniel utilizando TypeScript, com foco em criar uma presença online moderna, responsiva e profissional para a clínica.

---

## 📌 Sobre o projeto

Este projeto tem como objetivo apresentar a Clínica Peniel de forma digital, permitindo a divulgação de informações, serviços e facilitando o contato com pacientes.

Foi desenvolvido com foco em boas práticas de desenvolvimento frontend moderno, utilizando tipagem com TypeScript para maior segurança e organização do código.

---

## 📌 Tecnologias utilizadas

- TypeScript
- React (provavelmente)
- HTML5
- CSS3
- Vite / Next.js (se aplicável)
- JavaScript (base)

---

## 📌 Funcionalidades

- Página institucional da clínica
- Apresentação de serviços
- Seção de contato
- Layout responsivo (mobile e desktop)
- Interface moderna e simples
- Componentização de telas (caso React)
